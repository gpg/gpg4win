These icons are used for Gpg4win and GnuPG Desktop.

The Windows build uses the following icons:
* Icons with sizes 16, 24, 32, 48, 64, 128, and 1024 are added to the
  executable as icon resource. (Icons with sizes 256 and 512 would be
  ignored in favor of the 1024 icon.)
* The SVG icon is embedded as Breeze icon.

The AppImage build uses the following icons:
* Icons with sizes 16, 32, 64, 128, and 256, and scalable icons are
  deployed as hicolor icons.
* The SVG icon is added as Breeze/Breeze Dark icon replacing the existing
  normal and symbolic icons.
